# 베타 수집 가이드 — 물리 탐지(31차, C안) 신호 사전 만들기

> 대상: 베타 테스터(정글러) · 목적: "학원 신호 사전"(SSID/기지국/좌표)을 모아 다음 배포에 사전 탑재
> 버전: v1.8.0부터 설정 → "물리 탐지 (베타)" 섹션이 보입니다.

## 1. 왜 모으는가

현재 각자 폰이 **자동 학습**으로 스스로 학원 신호를 익히지만,
테스터들이 모은 샘플을 합치면 개발자가 학원 공식 신호(층별 SSID/AP, 주변 기지국 목록, 정확한 지오펜스 좌표)를
**초기값으로 앱에 내장**할 수 있습니다. 그러면 신규 설치자도 학습 기간 없이 바로 "학원 안/밖" 판정이 가능합니다.

## 2. 테스터가 할 일 (2~3일, 총 10분)

1. 설정 → **물리 탐지 (베타)** → `학원 근처 감지` 켜기 → 위치 권한 요청이 뜨면 **허용**
   - 수집 모드는 이 토글과 함께 기본으로 켜집니다 (끄고 싶으면 아래 토글로 개별 OFF)
2. **학원에 있을 때**: 설정에서 `지금 위치를 학원으로 학습` 1회 누르기
   (현재 연결 Wi-Fi·주변 기지국·좌표가 학원 신호로 등록됨)
3. 평소대로 사용 (학원 출입/외출/귀가). 5분 틱마다 샘플이 폰 안에 자동 누적됩니다
4. (선택) `즉시 감지(지오펜스)` 켜기 — 반응속도가 빨라집니다. 위치 권한을 "항상 허용"으로 바꾸라는 안내가 뜨면 앱 설정 화면에서 변경
5. **전송은 2가지 방법 중 편한 쪽으로** (2~3일 후):
   - **(자동, v1.9.0~)** 켜두면 **매일 1회** + **버퍼 1200건 도달 시 1회** "📧 수집 데이터 전송 준비됨" 알림이 뜹니다 → 탭하면 첨부가 완료된 메일 작성 화면이 열립니다. 설정에서 `수신 이메일`을 미리 입력해 두면 받는 사람이 채워져 있어 **전송만 누르면** 됩니다
   - **(수동)** 설정 → `수집 데이터 내보내기` → 공유 시트로 개발자(@giyeop-cody)에게 전달해 주세요

## 3. 무엇이 수집되는가 (프라이버시 고지)

| 항목 | 내용 | 용도 |
|------|------|------|
| 연결 Wi-Fi SSID·BSSID | 학원/집/카페 등 연결 기록 | 학원 신호 후보 식별 |
| 기지국 ID | LTE/5G 셀 식별자 목록 | Wi-Fi 안 쓰는 사용자용 근처 판정 |
| 좌표(선택) | "학원으로 학습" 누른 순간 1걸만 | 지오펜스 중심 후보 |
| 판정 상태 | 학원 안/밖, 점수, 시각 | 오탐/미탐 분석 |

**원칙**
- 샘플은 **휴대폰 안에만 저장**됩니다. 서버 업로드나 익명 전송은 없습니다.
- 외부로 나가는 유일한 경로는 사용자가 직접 공유·전송을 확정할 때뿐입니다 (자동 전송 알림도 직접 탭·전송해야 기기 밖으로 나갑니다).
- SSID·BSSID·셀 ID는 위치를 역추적할 수 있는 정보입니다. 내보내기 시 개발자가 외부에 재배포하지 않으며, 신호 사전 추출 후 원본은 폐기합니다.
- 언제든 설정에서 물리 탐지를 끄고, 앱 데이터 삭제로 수집물을 완전 제거할 수 있습니다.

## 4. 개발자가 받아서 하는 일

1. 여러 테스터의 JSON(이메일·공유 시트로 경유)을 취합 → 빈도/지속성 기준으로 **학원 공식 신호 목록** 추출
   (예: 3명 이상 공통 SSID, 학원 시간대 반복 등장 셀 ID)
2. 지오펜스 좌표 중간값 산출
3. 다음 배포에 `phy_seed` 초기값으로 내장 → 신규 사용자 즉시 판정 가능
4. 원본 내보내기 파일은 추출 직후 폐기

## 5. 판정이 이상할 때 (오탐/미탐 신고)

- 카페에서 "퇴실 누락"이 울리거나, 학원인데 "학원 밖"으로 나오면:
  설정 → 물리 탐지 상태 줄(판정/학습 건수) 스크린샷 + 진단 로그(PHY 태그)를 함께 공유해 주세요.
- 학습 데이터를 초기화하고 싶을 때: 설정에서 감지 OFF → `지금 위치를 학원으로 학습`으로 재시작, 또는 앱 데이터 삭제.
